**[P] AI Finance Assistant - Open Source Financial Analysis Platform**

I've built a comprehensive AI-powered financial analysis platform that I'd love to share with the community.

**Key Features:**
- Real-time stock data integration (Polygon.io API)
- AI-powered analysis using AWS Bedrock Claude Sonnet
- Advanced technical indicators (RSI, MACD, Moving Averages, Bollinger Bands)
- Machine learning trading signals
- Interactive web interface built with Streamlit
- Docker containerization for easy deployment

**Tech Stack:**
- Python 3.10+
- Streamlit for the web interface
- AWS Bedrock for AI capabilities
- Polygon.io for market data
- Docker for deployment
- Comprehensive CI/CD pipeline

The platform is designed for traders, financial analysts, and anyone interested in AI-driven market analysis. It provides both technical analysis and AI-powered insights to help make informed trading decisions.

**Repository:** https://github.com/vanhoangkha/ai-finance-assistant
